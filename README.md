# 项目描述

本项目实现了一个map容器       （类似于c++ STL 中的map）

* 底层数据机构是红黑树

## 支持功能：

* 插入键值对，
* 查找键值对，
* 打印键值对，
* 中序遍历整棵树，

例子见`main.cpp`
